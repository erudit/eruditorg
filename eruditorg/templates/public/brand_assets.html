    {# Colour palette #}

    {% trans 'Couleurs et illustrations' %}

    {% trans 'La charte graphique d’Érudit repose sur l’utilisation de 3 couleurs: le rouge, le bleu et le gris.' %}

      {% trans 'Rouge' %} {% trans 'Bleu' %} {% trans 'Gris' %}
    RGB 255 66 66 10 36 99 247 247 247
    HEX #ff4242 #0a2463 #f7f7f7
    PANTONE 1788 2738
    {% blocktrans %}

    Ces couleurs sont utilisées dans nos illustrations et respectent les règles suivantes :

    1. La portion de vide (blanc) doit idéalement représenter environ ½ à ⅓ de la surface.
    2. Il doit toujours y avoir une prédominance de rouge par rapport au bleu dans les compositions.
    3. Lorsqu’un visuel autre que les formes abstraites est requis, utiliser un aplat de rouge.
    {% endblocktrans %}

    {% blocktrans %} Les illustrations sont disponibles sur demande. {% endblocktrans %}

    {# Font and typography #}

    {% trans 'Police' %}

    {% blocktrans %} Érudit utilise la police de caractère Maax pour ses communications imprimées et numériques, en deux graisses : « regular » et « bold ». {% endblocktrans %}

    Maax regular Maax bold
    echo echo
    romeo romeo
    uniform uniform
    delta delta
    india india
    tango tango

    {% blocktrans %} Lorsque l’utilisation de cette police n’est pas possible, utiliser une police linéale (« sans sérif ») : Arial, Helvetica. {% endblocktrans %}
